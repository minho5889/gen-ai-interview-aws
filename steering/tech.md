## 0. Mental model: Strands is the framework; the model provider is swappable

| Layer | What it is | This project |
|---|---|---|
| **Framework** (in-process) | The Python agent loop, tools, hooks, multi-agent topology | **Strands SDK** |
| **Model provider** | The LLM Strands calls — swappable, one class | **`GeminiModel`** (Gemini API, `GOOGLE_API_KEY`) |

> **Key concept for Strands beginners:** "Should I use Bedrock or Strands?" is a category error.
> Strands is always the framework. The question is which *model provider* you plug in:
> `BedrockModel` (needs AWS + IAM), `GeminiModel` (needs `GOOGLE_API_KEY`), `AnthropicModel`
> (needs `ANTHROPIC_API_KEY`), `OllamaModel` (fully local, no key). For this project we use
> `GeminiModel` — no AWS account needed, runs on your laptop today.

## 1. Strands core — the real API (verified, v1.42.0)

```python
from strands import Agent, tool
from strands.models import GeminiModel

# GeminiModel verified constructor (strands_agents==1.42.0, google-genai==1.75.0):
# - model_id and params are GeminiConfig keyword args passed as **kwargs
# - temperature / token limits go INSIDE params={} — NOT as top-level constructor args
# - API key: set GOOGLE_API_KEY env var (google-genai SDK default)
#   OR pass explicitly: GeminiModel(client_args={"api_key": "..."}, model_id=...)
model = GeminiModel(
    model_id="gemini-2.5-flash",
    params={"temperature": 0.1, "max_output_tokens": 1500},
)

@tool
def check_booking(booking_id: str) -> dict:
    """Fetch a travel booking record. The docstring IS the tool description the model sees."""
    ...

agent = Agent(
    model=model,
    system_prompt="You are ...",
    tools=[check_booking],
)
result = agent("Is my hotel booked for June 15?")  # synchronous
```

**Verified `Agent(...)` constructor knobs** (from `strands/agent/agent.py`):
`model`, `tools`, `system_prompt`, `hooks`, `session_manager`, `conversation_manager`,
`structured_output_model`, `callback_handler`, `trace_attributes`, `agent_id`, `name`,
`concurrent_invocation_mode`. These are real.

**All available model providers** (`strands/models/`): `bedrock`, `anthropic`, `gemini`,
`sagemaker`, `litellm`, `ollama`, `openai`, `mistral`, `writer`, `llamaapi`, `llamacpp`.
`OllamaModel`/`LiteLLMModel` are great for offline testing with no API key.

## 2. Multi-agent: pick the primitive on purpose

Strands gives you **four** ways to compose agents (`strands/multiagent/`):

| Primitive | Import / shape | Use when | Don't use when |
|---|---|---|---|
| **Agents-as-tools** | a `@tool` fn that internally builds + calls a sub-`Agent` | A parent needs an *occasional* specialist | You need guaranteed parallelism or a fixed DAG |
| **Graph** | `from strands.multiagent import GraphBuilder` → deterministic DAG | Fixed control flow, **parallel fan-out/fan-in**, conditional edges | Routing is genuinely open-ended |
| **Swarm** | `from strands.multiagent import Swarm` → autonomous peer handoff | Agents hand off dynamically | You want determinism / auditability |
| **A2A** | `strands.multiagent.a2a` | Cross-process agents over A2A protocol | Everything is in one process |

> **This project uses `GraphBuilder`** — three reviewer nodes with no edges between them run
> in parallel, all converging on a synthesis node. That's structure, not a model decision.

## 3. Tools, MCP, and the prebuilt toolbox

- `strands_agents_tools` ships real tools (`.venv/.../strands_tools/`). Relevant here:
  `mcp_client`, `http_request`, `file_read`, `handoff_to_user`.
- **MCP is consumed via `mcp_client`** — not hand-written JSON-RPC. This project's MCP servers
  (`fetch`, `filesystem`) run as local `stdio` processes registered in `.mcp.json`.
- **Never let the model call free-form HTTP.** Tools expose *typed, parameterized* operations.
  The `@tool` signature + docstring is the contract the model sees.

## 4. Safety / guardrails — the idiomatic pattern

**Tool-call enforcement = a `HookProvider`** (verified: `BeforeToolCallEvent` exists in
`strands/hooks/events.py` with fields `selected_tool`, `tool_use`, `invocation_state`, `cancel_tool`):

```python
from strands.hooks import HookProvider, HookRegistry
from strands.hooks.events import BeforeToolCallEvent

class ReadOnlyGuard(HookProvider):
    BLOCKED = ("write", "delete", "update", "put_", "create", "drop", "remove", "cancel")

    def register_hooks(self, registry: HookRegistry) -> None:
        registry.add_callback(BeforeToolCallEvent, self._enforce)

    def _enforce(self, event: BeforeToolCallEvent) -> None:
        name = event.tool_use["name"].lower()
        if any(verb in name for verb in self.BLOCKED):
            # Set cancel_tool — returns a clean tool-result to the model so it can recover.
            # DO NOT raise — that kills the whole loop.
            event.cancel_tool = f"DENIED: '{name}' is a mutating op; verifier is read-only."

agent = Agent(model=model, tools=[...], hooks=[ReadOnlyGuard()])
```

## 6. Quick "hallucinated → real" cheat sheet

| ❌ Wrong | ✅ Real |
|---|---|
| `mlflow.strands.autolog()` | `StrandsTelemetry().setup_console_exporter()` or `.setup_otlp_exporter()` |
| `@agent.on(BeforeToolCallEvent)` decorator | `HookProvider` + `registry.add_callback(...)` + `event.cancel_tool` |
| `raise SecurityException(...)` to block a tool | `event.cancel_tool = "<reason>"` (recoverable) |
| "agents-as-tools, run in parallel" | `GraphBuilder` parallel nodes → join |
| `GeminiModel(model_id=..., temperature=0.1, max_tokens=1500)` | `GeminiModel(model_id=..., params={"temperature": 0.1, "max_output_tokens": 1500})` |
| `GEMINI_API_KEY` env var (strands) | `GOOGLE_API_KEY` (google-genai SDK default) |
| "Strands Evals" library | Doesn't exist — build an LLM-as-judge harness |
